# Submission App

## Setup

#### 1. Clone the repo -

```
git clone https://github.com/NotShrirang/Submission-App.git
```

#### 2. Install requirements -
```
pip install -r requirements.txt
```

#### 3. Run the backend server -
```
python manage.py runserver
```
